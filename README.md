### Steps to Run
1. Clone this repository
	```
	git clone https://github.com/piya03/react-user-dashboard.git
	```

2.  Run 
	```
	 yarn install
	```
3.  Run
	```
	yarn start
	```

### Technologies Used

 - Reactjs (hooks)
 - Tailwindcss
 - Momentjs
 - Font awesome
